入管提出用変換ツール v007

PC用。学生アプリv030の勤務記録Excelを読み込み、同じフォルダにある template.xlsx を自動で読み込んで、入管提出用Excelを作成します。学生情報・アルバイト内容も読み込みます。

準備

このHTMLと同じ場所に、元様式のExcelを template.xlsx という名前で置いてください。
GitHub Pagesで使う場合は、同じリポジトリ直下に index.html と template.xlsx を置きます。
